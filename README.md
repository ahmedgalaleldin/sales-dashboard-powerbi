# 📊 Sales Dashboard (Power BI)

## 🔍 Overview
Interactive sales dashboard built using Power BI.  
This project analyzes sales performance across countries, products, and time.

## 📈 Features
- Total Sales, Profit, and Units Sold KPIs
- Sales by Country and Product
- Monthly Sales Trend
- Interactive filters
- Drill-through analysis
- Executive insights
